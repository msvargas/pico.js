# pico.js

Implements NPM package of pico.js library development by: [tehnokv/picojs](https://github.com/tehnokv/picojs)

## Install

```
npm install pico.js
```

```
yarn add pico.js
```

## Features

- Fast rapid face detection - Viola Jones 
- Cascade method
- Pupil detection
- Low CPU consumption
